# Short Text Emotion Classification Using AI

## Overview

This project uses machine learning to classify emotions in short text.

Possible emotions:
- Joy
- Sadness
- Anger
- Fear
- Love
- Surprise

## Dataset

DAIR-AI Emotion Dataset

## Model

Logistic Regression with TF-IDF text vectorization.

## Results

The model was trained on thousands of labeled text examples and evaluated on a separate test set.

## Example

Input:

> I am excited to start my summer program.

Prediction:

> Joy

## Technologies

- Python
- Google Colab
- Scikit-Learn
- Pandas
- Hugging Face Datasets
